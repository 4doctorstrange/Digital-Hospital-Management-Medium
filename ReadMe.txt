A CLI program that maintains and update the record of patients.

To execute program follow the given steps:
1-Put all the files i.e Personal_data.py,Record.py,Main.py,Input.py,Medical_data.py  in single folder.
2-Compile Personal_data.py
3-Compile Medical_data.py
4-Compile Input.py
5-Compile Record.py
6-Compile Main.py
7-If above steps are executed correctly than user just have to answer queries asked in program.
8-You can also see a file name 'database.db' created in the same folder.

Prerequisites:
1-Python3 should be installed in system.
2-Compiling python program through Command Line
